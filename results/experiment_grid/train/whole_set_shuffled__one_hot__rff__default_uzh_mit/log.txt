======================================================================
EXPERIMENT LOG
======================================================================

'Aggregation' means the peptide clumped together during synthesis
instead of staying dissolved -- something chemists want to predict
and avoid. This experiment checks how well a model can predict, from
the peptide and its synthesis conditions, whether aggregation happens.

Data split:              whole_set_shuffled
  -> Same as 'whole peptide', but the order of amino acids in each peptide was randomly shuffled. This is a sanity check: if the model still performs well on shuffled peptides, it is probably not learning anything meaningful about the real sequence.

Data representation:    one_hot
  -> Represents each amino acid as a simple marker for which of the ~20 amino acids it is.

Prediction model:        rff
  -> Random Forest -- a large committee of decision trees that vote on the answer.

Protecting-group scheme: default_uzh_mit
